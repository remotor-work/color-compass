# Color Compass

Небольшое статическое JavaScript-приложение для демонстрации цветового круга и популярных схем подбора цветов.

## Что внутри

- выбор базового цвета через `HEX`, нативный пикер и ползунки `HSL`
- схемы: монохромная, аналоговая, комплементарная, разделённо-комплементарная, триада, квадрат, тетрада
- визуализация точек на цветовом круге
- готовая палитра с `HEX` и `HSL`
- копирование цвета в буфер обмена

## Запуск локально

Приложение не требует сборки. Достаточно открыть `index.html` в браузере.

Если нужен локальный сервер:

```powershell
python -m http.server 4173
```

После этого откройте `http://localhost:4173`.

## Деплой

Самый быстрый вариант без сервера:

1. создать репозиторий на GitHub
2. залить эти файлы в ветку `main`
3. включить GitHub Pages: `Settings -> Pages -> Deploy from a branch`
4. выбрать ветку `main` и корень `/ (root)`

После этого приложение будет доступно как статический сайт.
